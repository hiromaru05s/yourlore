# LORE — 수익 구조 (확정 스펙)

> 2026-07-03 확정. 원칙: **파워는 절대 팔지 않는다** (인게임 덱빌딩 = 완전 공정 대전이 LORE의 정체성).
> 수익원은 광고 + 코스메틱 + 구독뿐이다.

## 1. 광고 (무료 유저)

| 지면 | 규칙 |
|---|---|
| 상시 배너 | 게임 화면 중 상시 노출 (갱신 30~60초) |
| 전면 광고 | **3판당 1회** (매판 아님 — 리텐션 보호) |
| 보상형 광고 | **하루 3회**, 시청 1회 = 가챠 1회 |

- 구독자는 전 광고 제거 (보상형 포함).
- 포털(CrazyGames 등) 입점 빌드는 포털 SDK 광고 정책을 따름 → 자체 광고/외부결제 금지가 보통. **포털 빌드와 자체 도메인 빌드 투트랙.**

## 2. 크레딧 (단일 소프트 커런시)

- 가챠 1회 = 크레딧 **2~100 랜덤**, 고액은 희박 (기대값 ~12로 튜닝, 확률표는 확정 후 공개 페이지에 게시)
- 사용처: 카드 프로텍터(슬리브), 필드 디자인, 이모티콘

| 상품 | 가격 |
|---|---|
| 노말 디자인 (슬리브/필드) | 1,000 |
| 레어 디자인 (슬리브/필드) | 2,000 |
| 이모티콘 | 미정 (TBD) |

## 3. 무료 진행 보상 (첫 구매 동선)

- 봇전 4모드: 쉬움 / 보통 / 어려움 / 지옥
- 쉬움+보통+어려움 첫 격파 합계 **~600 크레딧**
- 랭크 신규 티어 도달 시 소액 크레딧 (제안: 초반 티어당 ~100, 수치 튜닝 대상)
- **지옥 첫 격파 = 비매품 전용 스킨** (크레딧 아님 — 돈으로 못 사는 위신 아이템)
  - 지옥 봇 = 클레어보이언트 봇 재활용 ("이 봇은 당신의 패를 봅니다" 명시 라벨)

설계 목표: 무료 유저가 **3~7일 안에 첫 스킨(1,000)** 획득. 이후 F2P 수입은 일 ~36크레딧(보상형 3가챠)으로 다음 스킨까지 ~1개월 — 첫맛 빠르고 이후 느린 표준 커브.

## 4. 유료 (SKU 3개만)

| SKU | 가격 | 내용 |
|---|---|---|
| **구독 (단일 플랜)** | **$7/월** | 전 광고 제거 + 매월 1,000크레딧 즉시 + 매일 가챠 5회 |
| 크레딧 팩 S | $4 | 1,000 크레딧 |
| 크레딧 팩 L | $7 | 2,000 크레딧 |

- 구독 월 가치 ≈ 1,000 + 가챠 150회(기대 ~1,800) ≈ **2,800크레딧 + 광고제거 = $11+ 상당을 $7에** → 구독이 항상 최고 가치 (의도된 설계).
- 광고제거 단독 상품 없음 — 광고제거는 구독에만 묶어 구독 가치를 유지 (1회성 판매는 장기 구조 훼손 우려로 기각).
- 크레딧 팩은 즉시성 니즈용 진입 사다리 ($4 첫 결제 → $7 구독 전환).

## 5. 예상 수익 (1,000 DAU 기준)

| 원천 | 월 추정 |
|---|---|
| 광고 (배너+전면+보상형, 애드블록 ~30% 감안) | $150~400 |
| 구독 (MAU 3~5천 × 전환 0.5~1.5% = 20~60명 × $7) | $140~420 |
| 크레딧 팩 | 소액 |
| 서버비 (Workers Paid, hibernation 적용) | −$5~10 |
| **합계** | **~$300~800/월** |
